# GamersNote

ゲームに関する記事を投稿できるメディアプラットフォーム「GamersNote」を開発しました。

![GamersNoteの画像](app/static/ogp.png)

## GamersNoteの概要

GamersNoteはゲームに関する記事を投稿できるメディアプラットフォームです。[note](https://note.com/)をゲームに特化させたら面白いのではないかと思い作りました。現時点で実装されている機能は以下の通りです。

- ユーザー登録
- ログイン
- セッション管理
- 記事の閲覧・投稿・編集・削除
- 記事にいいねをつける
- 記事にコメントする
